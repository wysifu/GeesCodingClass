august

Students will ease into programming by first learning about the Internet and computers.


week 1 - The Internet

Daily Objective - Students will start by getting familiar with what the Internet is, how it works, and all the components that are involved in creating the Internet. We will use in-class time to go through the Internet crash course on gcfglobal. Each student will be expected to take notes as they go through the crash course to remember concepts later. Complete 2-3 lessons on gcfglobal with in-depth notes taken in notebook detailing interesting facts, important things to remember, key vocalbulary terms, etc.

week 2 - The Internet

Daily Objective - Students will continue the Internet crash course and class discussions will be held for students to attempt to verbally demonstrate understanding of key concepts and topics. A multiple choice quiz will be given to monitor which concepts are being understood versus which topics may need more time and attention. Complete 2-3 lessons on gcfglobal with thorough notes taken in notebook detailing interesting facts, important things to remember, key vocalbulary terms, etc. Participate and engage with others in class discussion about the Internet; what is it ? how does it work ? how do we access the Internet ? what can we do online ? etc.

week 3 - Computers

Daily Objective - Students will follow up the information on the Internet crash course with the basic computers crash course, also provided on gcfglobal. It details how individual computers work, the components they are composed of, and offers a good foundation of knowledge on software and hardware of various device types, including but not limited to desktops, laptops, and smartphones. We will use in-class time to go through the course and take notes along the way. Complete 2-3 lessons on gcfglobal with in-depth notes taken in notebook detailing interesting facts, important things to remember, key vocalbulary terms, etc.

week 4 - Computers

Daily Objective - Students will continue the computer basics crash course and class discussions will be held for students to attempt verbally demonstrating their understanding of key concepts and topics. A multiple choice quiz will be given to monitor which concepts are being understood versus which topics may need more time and attention. Complete 2-3 lessons on gcfglobal with through notes taken in notebook detailing interesting facts, important things to remember, key vocalbulary terms, etc. Participate and engage with others in class discussion about the computers themselves; which components of the computer do what, different types of computers, how do computers follow commands, etc.


september

Students will shift focus to programming languages, beginning with HTML5.

week 1 - HTML Tags

Daily Objective - Students will begin learning HTML by signing up on freecodecamp and beginning the detailed HTML course. Each student will be expected to take notes and complete the lessons. Additional material from codeacademy pro will be used in whiteboard demonstrations and live code demonstrations via screen mirror or zoom. Homework will consist of completing a few lessons and taking detailed notes. The following day in class will have in class discussion about current topics and concepts. In-class activities will have students perform concepts to solidify what was learned in lessons on previous day with real experience.

week 2 - HTML

Daily Objective - Students will resume the freecodecamp HTML course and practice taking detailed notes. Additional material from codeacademy pro will be used for extra information if students are not completely grasping concepts. Students will practice HTML concepts in text editors to use newly aquired programming skills as they continue to complete the HTML course and take notes for themselves.

week 3 - HTML Semantic Tags

Daily Objective - Students will continue through the freecodecamp HTML taking detailed notes. Additional material from codeacademy pro will be used for extra information if students are not grasping concepts. Students will continue practicing HTML concepts in text editors to use newly aquired programming skills. Students will create a single webpage using only HTML to make an About Me page.

week 4 - HTML

Daily Objective - Students will work in-class to create a multi-page webstite using only HTML that contains internal links that link back and forth between the webpages. Students will write a fictional story on one page; 'Oh That Story', their story; 'My Story', and someone else's story; 'her/his story'


october

Students will begin incorporating cascading style sheets into their web development repertoire to give their HTML code to give a unique look.

week 1 - CSS

Daily Objective - Students will begin learning CSS through the freecodecamp CSS course. Students will be expected to take detailed notes and implement new knowledge by concurrently coding in a text editor. Additional material from codeacademy pro will be used to teach students via whiteboard or zoom demonstrations. Homework will consist of students completing a few lessons and taking detailed notes. The following day will consist of student guided in-class discussion about current topics and concepts. In-class activities will propel students to perform concepts to solidify what was learned in lessons on previous day with real experience.

week 2 - CSS Selectors and Parameters

Daily Objective - Students will resume the freecodecamp CSS course and practice taking detailed notes. Additional material from codeacademy pro will be used for extra information if students are not fully grasping concepts. Students will practice CSS concepts in text editors to use newly aquired programming skills as they continue to complete the CSS course and take notes for themselves.

week 3 - CSS Classes and ids

Daily Objective - Students will continue through the freecodecamp CSS taking detailed notes. Additional material from codeacademy pro will be used for extra information if students are not grasping concepts. Students will continue practicing CSS concepts in text editors to use newly aquired programming skills. Students will add to their HTML single webpage using CSS to give their About Me page a unique look.

week 4 - CSS

Daily Objective - Students will work in class to add to their multi-page HTML website by using CSS to style all the pages the same, or create a wacky website with different designs for each page.

november

Students will upload their websites on github pages for free then begin to learn more about the nature of web development.

week 1 - Command Line

Daily Objective - Students will be guided through codeacademy pro command line course to get a clear understanding of how to communicate directly to the computer.

week 2 - Front-End Web Development

Daily Objective - Students will dive deeper into front end web development by learning about making websits accessible, so everyone can enjoy our content and responsive, so our websites can look good on all device types.

week 3 - Javascript

Daily Objective - Students will begin learning basic Javascript.

week 4 - Javascript

Daily Objective - Students will begin learning basic Javascript.
