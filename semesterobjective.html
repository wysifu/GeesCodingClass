Semester Objective

  • components of computer

    • - students should have a basic knowledge on the hardware/software components of the computer and their functions.


  • internet, servers, browser

    • - students should have a basic knowledge on a few compoent of the worldwide web and how computers communicate with other computers through the internet.

  • basic html syntax

    • - students should be able to demonstrate overall understanding of simple html syntax.


  • intermediate html syntax

    • - students should be able to demonstrate knowledge of intermediate html topics such as heirarchy, html semantics, forms, and buttons.


  • html tags

    • - students should be able to demonstrate an overall knowledge of a handful of html tags off memory, and they should know where to go online to find documentation of all html tags if needed.


  • basic css

    • - students should be able to demonstrate overall understanding of simple css concepts and topics.


  • intermediate css

    • - students should be able to demonstrate knowledge of intermediate css topics such as selectors, classes, ids, parameters, and how they work within the css environment.


  • css links, stylesheets, and style tags

    • - students should be able to demonstrate knowledge on css stylesheets that use link tags to link into html code, and style tags used within html documents and inline html text.


  1. students will create a simple one page webpage using html

  2. students will style the one page website using style tags within html

  3. students will create a simple website with multiple pages using only html

  4. students will style the multiple page website by linking a css stylesheet in the html pages

  5. students will share projects on the internet for free using github for others to see

  6. students should understand basic javascript concepts
