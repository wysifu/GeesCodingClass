project-based learning

We can learn alot by reading text, watching videos, and completing courses but the best way to retain knowledge is to complete projects that simulate real world situations, problems, or objectives. Most of this intro to programming class will be putting the knowledge learned in the online courses and in class discussion to the test during in class and individual student projects. Projects will ultimately test the students' resourfulness, problem solving abilities, memory, and creativity to create something with real world application. Project based learning makes more of an impact because students are no longer just studying concepts, theory is being put into action by real time application.

individualized instruction

Individualized instruction can often lead to understanding as much as or even more than group activities can. With the use of personal whiteboards, students can work through problems one on one with the instructor or another student.

core class integration

Core subjects will be incorporated when students are programming. In the first semester, mostly English and grammar will be used in conjunction with computer programming, where students will be writing webpages on various topics. As students become more familiar with programming languages, core math and science topics will be integrated. Javascipt and Python specifically has the ability to do data analysis. Science and math classes can collect data and pass it to the coding class where students will intake the data and provide some sort of analysis using graphs and more.
